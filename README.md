# LLM-Nova
Custom variant inheriting from llama3.2.


## Create custom model using ollama

Clone the repo
```
git clone https://github.com/FullstackCodingGuy/LLM-Nova.git
cd LLM-Nova
```

Create model

```
ollama create Nova -f ./Modelfile
```

To test the model

```
ollama run Nova
```
